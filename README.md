# GEP Financials

Dashboard para el seguimiento y gestión de las finanzas del grupo de empresas GEP.

## Descripción

GEP Financials es una plataforma centralizada que permite visualizar, analizar y controlar los indicadores financieros clave de las empresas del grupo. Proporciona una visión consolidada en tiempo real del estado financiero, facilitando la toma de decisiones estratégicas.

## Funcionalidades previstas

- **Panel de control (Dashboard)** — Visión general del estado financiero consolidado del grupo
- **Ingresos y gastos** — Seguimiento detallado por empresa, categoría y período
- **Flujo de caja** — Control de tesorería y proyecciones de liquidez
- **Presupuestos** — Comparativa entre presupuesto y ejecución real
- **Informes financieros** — Generación de reportes en distintos formatos (P&L, Balance, etc.)
- **Alertas y KPIs** — Indicadores clave con umbrales configurables
- **Multiempresa** — Gestión independiente y consolidada de todas las sociedades del grupo
